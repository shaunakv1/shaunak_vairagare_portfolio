Hurricanes

Hurricanes is an interactive mapping application that easily searches and displays global tropical cyclone data. Users are able to query storms by the storm name, ZIP Code, city, state, geographic region, pressures, intensities and time range.

Key Features: Angular, Leaflet, HTML5, CSS3,Less, HighCharts, Grunt, JSON, ArcGIS Rest, Grunt

Sea Level Rise

Sea Level Rise Viewer displays potential future sea levels for coastal USA. A slider lets users quickly simulate water levels directly on map. This viewer also shows sea level rise on actual pictures of local landmarks which correlates with the sea level shown on map. Also provides features like models for marsh migration and socio-economic overlay. It simultaneously displays seven tile cache layers from ArcGIS Server on leaflet map. It uses multiple-buffering on client side to display smooth transitions of sea level changes.

Key Features: OpenLayers 4, HTML5, CSS3, JavaScript, SASS, jQuery, HighCharts, Grunt, jUnit, ArcGIS Rest, Tile Cache, GeoJSON

Lake Level Viewer

Lake Level Viewer is a new generation version of the Sea Level Rise Viewer below; accept it shows potential lake level changes for Great Lakes. It also shows 13 levels on map, for both rise and drop in the lake levels. Cached tiles are loaded from ArcGIS Server and rendered on leaflet map to show smooth water level change on map.

Key Features: Leaflet, HTML5, CSS3, Less, JavaScript, jQuery, HighCharts, Grunt, jUnit, ArcGIS Rest, Tile Cache, GeoJSON

Land Cover Atlas

Land Cover Atlas is an online map viewer that provides user-friendly access to regional land cover and land cover change information developed through NOAAâ€™s Coastal Change Analysis Program (C-CAP). The tool summarizes general change trends (such as forest losses or new development) and can highlight specific changes of interest. It uses huge datasets exposed using a combination of ArcGIS Server, JSON Web Services and WMS server to display statistics and change rasters on interactive OpenLayers map quickly and efficiently on a very user-friendly interface.

Key Features: OpenLayers, HTML5, CSS3, JavaScript, jQuery, HighCharts, Grunt, jUnit, Web Services, ArcGIS Rest, WMS, Tile Cache, JSON
